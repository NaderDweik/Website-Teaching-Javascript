JavaScript Strings: A Comprehensive Guide

Introduction to Strings in JavaScript

In JavaScript, text values are represented as strings. A string can contain letters, numbers, symbols, and spaces. Unlike some programming languages, JavaScript doesn't have a dedicated "char" type for single characters, instead treating them as strings of length one.

Strings in JavaScript are stored as a sequence of characters encoded using UTF-16, which ensures compatibility with many languages and special characters.

Different Ways to Define Strings

Strings can be created using different types of quotes:

let single = 'single-quoted';
let double = "double-quoted";
let backticks = `backticks`;

While single and double quotes are essentially interchangeable, backticks (`` ` ``) offer additional benefits, like multi-line strings and embedded expressions.

Backticks (Template Literals)

Backticks allow embedding expressions inside strings, which is useful for string interpolation.

let sum = (a, b) => a + b;
alert(`1 + 2 = ${sum(1, 2)}`);
// 1 + 2 = 3

Another advantage of backticks is multi-line support:

let guestList = `Guests:
* John
* Pete
* Mary`;

Escape Characters and Special Characters

Strings support various escape sequences, including:

  • \n - New line
  • \t - Tab
  • \\ - Backslash
  • \", \', \` - Escape quotes

Note: To insert an actual backslash in a string, you need to escape it by using \\.

Example of a string with a newline character:

let text = "Hello\nWorld";
alert(text); // Outputs: Hello (new line) World

String Length and Accessing Characters

The length property gives the number of characters in a string:

let str = "Hello";
alert(str.length); // 5

To access individual characters, you can use either bracket notation or the at() method:

let str = "Hello";
alert(str[0]); // H
alert(str.at(0)); // H

String Immutability

Strings in JavaScript are immutable, meaning you cannot modify a string's characters directly:

let str = 'Hello';
str[0] = 'h'; // Error: Cannot assign to read-only property

Instead, you can create a new string:

let str = 'Hello';
str = 'h' + str.slice(1);
alert(str); // hello

Searching and Modifying Strings

To search for a substring within a string, use methods like indexOf, includes, startsWith, and endsWith.

let str = "JavaScript is awesome!";
alert(str.indexOf('Java')); // 0
alert(str.includes('Script')); // true
alert(str.startsWith('Java')); // true
alert(str.endsWith('awesome!')); // true

String Comparison

Strings are compared character by character based on their UTF-16 values:

alert('a' > 'Z'); // true
alert('Apple' > 'apple'); // false
